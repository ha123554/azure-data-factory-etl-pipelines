# ARM Templates

This folder contains the Azure Resource Manager (ARM) templates for the Azure Data Factory project.

## Files

- **ARMTemplateForFactory.json** – Defines the Azure Data Factory resources, including pipelines, datasets, linked services, and triggers.
- **ARMTemplateParametersForFactory.json** – Contains the parameter values required to deploy the ARM template.

## Purpose

These ARM templates can be used to deploy or recreate the Azure Data Factory environment in another Azure subscription or resource group.
